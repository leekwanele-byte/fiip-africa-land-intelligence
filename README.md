Fountains Farm Integrated Investment Program (FIIP)

Conservation-First Land Intelligence Platform for Africa

Satellite → Data → Field → Community → Value

---

Overview

Africa does not lack resources — it lacks accessible intelligence to manage them sustainably.

The Fountains Farm Integrated Investment Program (FIIP) is a real-world, field-tested system built in Zimbabwe that combines satellite data, machine learning, and mobile tools to transform land into a multi-income, conservation-driven asset.

This system integrates:

- Gold exploration (reef + alluvial targeting)
- Vegetation monitoring and restoration
- Soil intelligence and productivity mapping
- Tree inventory and controlled clearing
- Grass and acacia-based silage systems
- Mobile data collection for real-time decision-making

All outputs are usable directly in the field via smartphone.

---

Real-World Validation (Shangani, Zimbabwe)

- 200-hectare farm analysed and digitised
- NW–SE structural trend identified (reef orientation)
- Multiple high-priority gold zones mapped
- Gold recovered from a satellite-predicted zone
- Vegetation and biomass systems integrated
- Field workers actively using mobile tools

---

Why This Matters

Across Africa, land is often:

- under-measured
- under-managed
- and historically exploited without local benefit

FIIP addresses this by:

- Bringing satellite intelligence to ground level
- Enabling community participation through mobile tools
- Creating multiple aligned income streams
- Supporting conservation alongside productivity

Core Capabilities

Gold Prospectivity Mapping

- Iron oxide detection (gossans)
- Clay alteration mapping
- Vegetation stress indicators (NDVI decline)
- Structural lineament extraction
- Machine learning refinement

---

Vegetation & Land Management

- NDVI (vegetation health)
- Moisture detection (MNDWI)
- Vegetation decline tracking
- Tree inventory and culling prioritisation

---

Soil Intelligence

- Organic carbon estimation
- Soil pH and texture mapping
- Integration with global soil datasets

---

Biomass & Silage Systems

- Grass veld monitoring
- Acacia utilisation (silage potential)
- Biofuel production (firewood & briquettes)

---

Mobile Data Collection

- Google Forms for field input
- GPS + photo capture
- AI-assisted plant identification
- Centralised cloud database (Supabase)

---

System Architecture

- Satellite Data: Sentinel-2, ASTER, SRTM
- Processing: Python + Google Earth Engine
- Database: Supabase (PostgreSQL)
- Field Tools: SW Maps + Google Forms
